# Funbox QT

- Typescript, React, PostCSS

## Требования

- [Node.js 12+](https://nodejs.org/)

## Установка

```bash
git clone https://github.com/0151/funbox-qt.git --depth 1 funbox-qt
cd funbox-qt
npm i
```

## Использование

### Сборка проекта

```bash
npm run build
```

### Запуск окружения разработки

```bash
npm start
```
